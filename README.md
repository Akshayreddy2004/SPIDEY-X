# 🕷️ SPIDEY-X

SPIDEY-X is an intelligent AI assistant built using Python, Streamlit, Groq API, and SQLite.

## Features

* AI-powered conversations
* Persistent SQLite memory
* User memory storage
* Name recall
* Goal recall
* Chat history
* Clean Streamlit interface

## Tech Stack

* Python
* Streamlit
* Groq API
* SQLite
* Git & GitHub
